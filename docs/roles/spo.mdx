# Stake Pool Operator (SPO)

## Becoming an SPO

1. Download node [8.2.0-pre](https://github.com/input-output-hk/cardano-node/releases/tag/8.2.0-pre)
2. [Run the node](tutorials/start-a-node.mdx)
3. [Request funds from faucet](https://faucet.sanchonet.world.dev.cardano.org/basic-faucet)
4. [Register a pool](tutorials/register-spo.mdx)
5. [Request delegation from faucet](https://faucet.sanchonet.world.dev.cardano.org/basic-faucet)
