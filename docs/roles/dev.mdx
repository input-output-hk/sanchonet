# Developers

## Develop on SanchoNet

1. Run a node
2. Get funds from faucet
3. Build a cool tool to integrate with SanchoNet
4. Publish it to the community

### Voting Centers

Voting centers let you interact with the chain to view and vote on
specific proposals. They can also integrate with wallets to allow
delegation to DReps and submitting transactions to the wallets.
